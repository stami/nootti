# Nootti

## API speksit


| Route          | HTTP verb | Description |
| -----          | --------- | ----------- |
| /api/notes     | GET  | Get all titles |
| /api/notes     | POST | Create new note |
| /api/notes/:id | GET  | Get all data of a note |
| /api/notes/:id | PUT  | Update note (auto save) |
| /api/notes/:id | DELETE | Delete a note |
